# The Runge–Kutta method

Computing the approximate solutions of ordinary differential equations by the Runge–Kutta method.
_y' = y/(1+x) - 0.5*y^2_

_y(0) = 1_

pip install git+https://github.com/MargoILIrita/techpro.git@master
